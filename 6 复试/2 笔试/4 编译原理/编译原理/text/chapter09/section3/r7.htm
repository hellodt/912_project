（1）表示该符号布尔性质的属性域，可用1个bit位来表示；也可用1个布尔量来表示。如
defined 1 表示已定义
defined 0 表示尚未定义
或    
defined true 表示已定义
defined false 表示尚未定义
　　（2）表示符号的数据类型可以用若干个bit位来表示；也可用1个整型量来表示（C语言为例）
data-type 3个bit位
Char 000
Short 001
int 010
long 011
unsigned 100
float 101
double 110
或  
data-type 整型值
char 0
short 1
int 2
long 3
unsiqned 4
float 5
double 6

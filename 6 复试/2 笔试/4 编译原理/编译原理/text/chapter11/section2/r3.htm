我们从下面例子理解这些变换。
　　有四元式程序：
　　t1 := 4 - 2
　　t2 := t1 /2
　　t3 := a * t2
　　t4 := t3 * t1
　　t5 := b + t4
　　c := t5 * t5
　　进行合并已知量变换后得到：
　　t1 := 2
　　t2 := t1 /2
　　t3 := a * t2
　　t4 := t3 * t1
　　t5 := b + t4
　　c := t5 * t5
　　再进行复写传播和删除无用赋值等变换后得到：
　　t2 := 2 /2
　　t3 := a * t2
　　t4 := t3 * 2
　　t5 := b + t4
　　c := t5 * t5
　　再进行合并已知量变换后得到：
　　t2 := 1
　　t3 := a * t2
　　t4 := t3 * 2
　　t5 := b + t4
　　c := t5 * t5

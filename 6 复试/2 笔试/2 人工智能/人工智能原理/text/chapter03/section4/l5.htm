3.4.3 推理计算
1）"与"的计算：
　f1 (A1∧A2) = min { f1 (A1), f1 (A2)}
2）"或"的计算：
　f1 (A1∨A2) = max { f1 (A1), f1 (A2)}
3）"非"的计算：
　CF(～A ) = －CF(A )
4）由f1 (A)， A → B， (c1, c2, …, ck)。求f1 (B)
　m({b1}, {b2}, …,{bk}) ==(f1 (A)c1, f1 (A)c2, …, f1 (A)ck)
　m (U) = 1-∑f1 (A)ci, i = 1, …, k
5）证据的组合，m1, m2在U上的合成
　对于同样的证据，由于来源不同，分别得到二个概率分配函数m1, m2。
　定义：m = m1⊙ m2
　规定：
　m(Φ) = 0;
　m(A) = K∑m1(X)m2(Y), 当X∩Y = A
　其中K-1 = 1 - ∑m1(X)m2(Y), 当X∩Y =Φ
　　　　　= ∑m1(X)m2(Y), 当X∩Y ≠Φ
　　　且K-1 ≠ 0，
　若K-1 = 0， 认为m1, m2矛盾。没有联合基本概率分配函数。

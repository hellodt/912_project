第2题
解： 1) 要证明P → ( Q → P )永真
　　～（Q→P）＝ ～P∧Q
　　建立子句集：
　　S = { ～P, Q, P}
　　进行归结：
　　Q
　　～P,
　　P
　　□

2) 要证明( P → ( Q → R )) → ((P → Q) → ( P → R))永真
　　即证明 ～(( P → ( Q → R )) → ((P → Q) → ( P → R)))为永假。
　　将其化为合取范式：
　　～(( P → ( Q → R )) → ((P → Q) → ( P → R)))
　　= ( P → ( Q → R ) ∧ ～((P → Q) → ( P → R))
　　= (～P ∨ ～Q ∨ R ) ∧ (P → Q) ∧ ～ ( P → R)
　　= (～P ∨ ～Q ∨ R ) ∧ (～P ∨ Q) ∧ P ∧ ～R
　　建立子句集：
　　S = {～P ∨ ～Q ∨ R, ～P ∨ Q, P, ～R }
　　进行归结：
　　1) ～P ∨ ～Q ∨ R
　　2) ～P ∨ Q
　　3) P
　　4) ～R
　　5) Q 　　　　　　（2）（3）归结
　　6) ～Q ∨ R　　　（1）（3）归结
　　7) R 　　　　　　（5）（6）归结
　　8) □　　　　　　（4）（7）归结

3) 要证明( Q → ～ P) → ((Q → P) → ～ Q)为永真
　　即证明 ～(( Q → ～ P) → ((Q → P) → ～ Q) )为永假
　　将其化为合取范式：
　　～(( Q → ～ P) → ((Q → P) → ～ Q) )
　　= ( Q → ～ P) ∧ (Q → P) ∧ Q
　　= (～Q ∨ ～P) ∧ (～Q ∨ P) ∧ Q
　　建立子句集：
　　S = { ～Q ∨ ～P, ～Q ∨ P, Q }
　　进行归结：
　　(1) ～Q ∨ ～P
　　(2) ～Q ∨ P
　　(3) Q
　　(4) ～P 　　　　（1）（3）归结
　　(5) P 　　　　　（1）（2）归结
　　(6) □

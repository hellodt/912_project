# 面试（300）

一般在第二天上午8:30进行学术面试，第三天上午8:30进行专业面试。

> 2020年疫情原因，若采取线上复试，则面试比重可能加大。

## 综合面试（20*3）

综合面试包括自我介绍（中英文皆可），中英文提问，针对简历提问，一篇计算机相关的英文文本现场口译。

### 自我介绍

约两到三分钟，注意掌握时间。

### 口语问答

#### 针对性问答

由准备的简历内容所确定。

#### 常规问答

兴趣爱好，活动经历……

### 口译（口语+翻译）

给了一段外文文献，先用英语读一半，剩下的一半看着英语说中文翻译。

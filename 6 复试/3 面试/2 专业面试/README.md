# 专业面试（80*3） 10~15min

> 至少准备考生自述+成绩单。

专业面试包括ppt自述，然后老师提问。

## PPT自述 3~5min

> 现场准备书面材料16份。
>
> PPT不宜过多，10张左右。
>
> 关于选材方面，必须选择准备过的内容。这会影响到提问环节。

包括个人学习情况、实践活动与获奖、学术成果、特长爱好、对报考专业的科研了解情况等。
